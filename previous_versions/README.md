# Previous Versions

This directory contains frozen historical snapshots of released Schiavinato Sharing protocol versions.

## Policy
- The repo root remains the live working tree for the current protocol.
- Each `previous_versions/vX.Y.Z/` directory is an immutable historical snapshot.
- Newer snapshots aim to group that release's archived `manual_spec`, `software_spec`, `test_vectors`, and `whitepaper` materials in one place.
- Older releases may be partial snapshots when only some historical materials are clearly recoverable in-repo.
